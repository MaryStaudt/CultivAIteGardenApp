# Approved Sources Only

This folder is for trusted CultivAIte source material only.

Do not add random internet articles here. Every file should be something you would be comfortable letting CultivAIte use when giving gardening advice to real users.

## Allowed Sources

Use sources like:

- USDA
- University Extension Offices
- Land-grant universities
- Peer-reviewed gardening or agriculture papers
- Expert-reviewed CultivAIte guides
- User-uploaded seed packet information, but only for that user's specific plant or variety

## Do Not Use

Do not include:

- Blogs
- Reddit
- Pinterest
- Random websites
- AI-written articles
- Unreviewed social media posts
- Product marketing pages unless the question is specifically about that product label

These are example names only. Do not add fake files with these names. Only add the real source document if you have it.

```text
trusted-sources/

```

## Before Adding A File

Ask:

1. Is this from USDA, a university extension office, a land-grant university, a peer-reviewed source, or an expert-reviewed CultivAIte guide?
2. Does it clearly say who published it?
3. Is it specific enough to help gardeners make a real decision?
4. Is it current enough to be useful?
5. Would I want CultivAIte to rely on this when answering a beta tester?

If the answer is no, do not put it in this folder.
