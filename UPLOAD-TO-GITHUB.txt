CultivAIte mobile and web update

This folder contains the full replacement set for the live CultivAIte project.

In your existing CultivAIte GitHub repository:
1. Open the repository's main page.
2. Choose Add file, then Upload files.
3. Drag the CONTENTS of this folder into the upload area. Upload these items at the repository's top level:
   - api folder
   - icons folder
   - src folder
   - index.html
   - manifest.webmanifest
   - package.json
   - sw.js
   - vercel.json
4. When GitHub asks whether to replace files with the same names, choose Replace.
5. Commit directly to main.

Do not upload the outer folder itself. Upload the items inside it.

After the commit, Vercel will redeploy automatically. Open the normal CultivAIte website, then do one hard refresh before testing.
